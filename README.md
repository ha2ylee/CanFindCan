# CanFindCan
# 시각장애인을 위한 캔음료 인식 애플리케이션
<br/>

# 💡 프로젝트 소개
- 시각장애인들에게 새로운 소통 방식을 제공하기 위해 애플리케이션을 구현하고자 함
- 점자 표기로는 종류의 구분이 불가능한 캔 음료의 구분에 목적을 둠 
- 카메라로 캔을 비추면 캔의 종류를 분석하고, 캔의 밑바닥을 촬영하면 유통기한을 음성으로 안내
